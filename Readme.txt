Description du jeu :
Le but du jeu est récuperer toutes les pommes d'un niveau en évitant les mines. 
Parfois le niveau contiendra des coeurs qui redonne de la vie.

Architecture code :
Les fonction géométrique sont dans leur .js assossié, le chargement des images de l'audio est fait a part.
La detection des collision aussi.

Difficultés rencontrées :
Manque d'imagination et de motivation

Points fort :
-Niveau générér aléatoirement et donc ilimité en théorie.
-Maniable a la souris

Point faible :
-Niveau générér aléatoirement
-Peu de contenu
-Trop proche du TP
